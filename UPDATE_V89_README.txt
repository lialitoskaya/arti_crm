v89 — manual reassign priority

Что уточнено и закреплено:
- Автоназначение первого ответившего работает только для неназначенных чатов.
- Ответственного можно вручную переназначить через поле "Ответственный" в открытом диалоге.
- Ручное переназначение всегда имеет приоритет и не перезаписывается автоназначением.
- В metadata_json добавляется маркер ручного переназначения для диагностики:
  _crm_assigned_manual=true
  _crm_assigned_manual_user_id
  _crm_assigned_manual_at

После установки:
1. Остановить CRM.
2. Распаковать архив в C:\crm_marketplaces с заменой файлов.
3. Запустить CRM.
4. Проверить /api/debug/version — должна быть v89_manual_reassign_priority_2026-06-18.

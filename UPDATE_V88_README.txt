v88 — auto assign first responder

Что добавлено:
- Если чат не назначен, первый сотрудник, который успешно отправил ответ клиенту из CRM, автоматически становится ответственным.
- Чат сразу попадает во вкладку «Мои чаты» этого сотрудника.
- В открытом диалоге поле «Ответственный» заполняется этим сотрудником после отправки ответа.
- Существующий ответственный не перезаписывается.
- Назначение происходит только после успешной отправки сообщения в маркетплейс.
- Добавлена переменная .env: CRM_AUTO_ASSIGN_FIRST_RESPONSE=true.

После установки:
1. Остановить CRM.
2. Распаковать архив в C:\crm_marketplaces с заменой файлов.
3. Запустить CRM.
4. Проверить /api/debug/version — должна быть v88_auto_assign_first_responder_2026-06-18.

Отключение автоназначения:
CRM_AUTO_ASSIGN_FIRST_RESPONSE=false

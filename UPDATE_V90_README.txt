v90 — in-app notifications

Добавлено:
- Таблица notifications.
- Центр уведомлений в левом меню.
- GET /api/notifications.
- POST /api/notifications/{id}/read.
- POST /api/notifications/read-all.

События:
- Новое входящее сообщение в назначенном чате — ответственному.
- Новое входящее сообщение в неназначенном чате — активным менеджерам/админам.
- Ручное назначение ответственным — новому ответственному.
- Новая задача — назначенному сотруднику.

Защита:
- dedupe_key не даёт создавать дубли при повторной синхронизации.
- Старые сообщения из backfill не создают уведомления.
- Порог свежести: CRM_NOTIFICATION_RECENT_MESSAGE_HOURS=24.

После установки:
1. Остановить CRM.
2. Распаковать архив в C:\crm_marketplaces с заменой файлов.
3. Запустить CRM.
4. Проверить /api/debug/version — должна быть v90_in_app_notifications_2026-06-18.

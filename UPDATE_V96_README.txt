v96 — notifications unanswered only

Исправлено:
- Уведомления по сообщениям теперь показываются только если чат реально ждёт ответа.
- Если после сообщения клиента уже есть более поздний ответ менеджера, уведомление автоматически помечается прочитанным.
- Старые unread-уведомления по чатам, где последний месседж уже не входящий, очищаются при запросе /api/notifications.
- Для одного чата оставляется только уведомление по текущему последнему входящему сообщению, старые уведомления этого чата скрываются.

Это решает кейс, когда маркетплейс уже показывает две галочки/прочитанный или отвеченный диалог, а CRM продолжала показывать уведомление.

После установки:
1. Остановить CRM.
2. Распаковать архив в C:\crm_marketplaces с заменой файлов.
3. Запустить CRM.
4. Проверить /api/debug/version — должна быть v96_notifications_unanswered_only_2026-06-18.
5. Обновить браузер Ctrl+F5.

v85 — analytics dashboard

Что добавлено:
- Новый раздел «Аналитика» в левом меню.
- Backend endpoint: GET /api/analytics/chats.
- Отдельный сервис app/services/analytics.py для расчётов, чтобы не раздувать main.py.
- KPI:
  - всего обращений в день/период;
  - среднее время ответа;
  - обращения до 10:00 и после 19:00;
  - фильтр по диапазону часов;
  - количество закрытых обращений;
  - самый активный час;
  - пиковые часы обращений;
  - разбивка по маркетплейсам.
- Добавлены SQLite индексы для ускорения аналитики.
- Добавлена документация docs/ANALYTICS.md.

Как считаем:
- «Обращение» = входящее сообщение клиента.
- Среднее время ответа = от входящего сообщения до ближайшего следующего исходящего ответа в том же чате.
- Закрытые обращения = чаты со статусом closed и последней активностью в выбранном периоде.

После установки:
1. Остановить CRM.
2. Распаковать архив в C:\crm_marketplaces с заменой файлов.
3. Запустить CRM.
4. Проверить /api/debug/version — должна быть v85_analytics_dashboard_2026-06-18.
5. Открыть раздел «Аналитика» в левом меню.

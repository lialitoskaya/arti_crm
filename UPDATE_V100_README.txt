v100 — analytics hourly drilldown

Что добавлено:
- В разделе «Аналитика» появился блок «Проверка расчёта по часам».
- Теперь можно видеть конкретные чаты и сообщения, которые попали в отчёт как первые обращения клиента за день.
- В строке есть дата/час, маркетплейс, клиент/чат, текст сообщения и кнопка «Открыть».
- По клику на строку часа в графике можно быстро отфильтровать детализацию на выбранный час.
- Добавлен список служебных сообщений, исключённых из аналитики.
- Новый API:
  GET /api/analytics/chats/drilldown

После установки:
1. Остановить CRM.
2. Распаковать архив в C:\crm_marketplaces с заменой файлов.
3. Запустить CRM.
4. Проверить /api/debug/version — v100_analytics_hourly_drilldown_2026-06-18.
5. Обновить браузер Ctrl+F5.

v95 — clickable notifications cleanup

Что изменено:
- Уведомление теперь открывается по клику на всю карточку.
- Убраны лишние кнопки «Открыть чат» и «Скрыть».
- Оставлен только крестик для закрытия уведомления.
- Исправлено поведение: после закрытия уведомление помечается как прочитанное и не должно бесконечно крутиться по кругу.
- В тостах показываются только новые уведомления текущей сессии, а не весь старый unread-список.

После установки:
1. Остановить CRM.
2. Распаковать архив в C:\crm_marketplaces с заменой файлов.
3. Запустить CRM.
4. Проверить /api/debug/version — должна быть v95_notifications_clickable_cleanup_2026-06-18.

Обновление v54 — Ozon Questions + WB safe debug

ВАЖНО: этот архив собран БЕЗ верхней папки crm_marketplaces.
Распаковывайте содержимое архива прямо в корень проекта:
  C:\crm_marketplaces\
Так файлы app\main.py, app\static\app.js и т.д. должны замениться напрямую.

После распаковки:
1. Остановите сервер CRM полностью.
2. Распакуйте архив в C:\crm_marketplaces с заменой файлов.
3. Проверьте свой .env: добавьте блок Ozon Questions из .env.example, не затирая токены.
4. Запустите сервер.
5. Откройте http://127.0.0.1:8000/api/debug/version
   Должно быть: build_version = v54_ozon_questions_wb_safe_root_2026-06-15
6. В браузере нажмите Ctrl+F5.

Проверка раздела вопросов:
- В левом меню должна появиться кнопка «Вопросы».
- Нажмите «Вопросы» -> «Обновить».
- Если список пустой или ошибка: откройте /api/debug/ozon/questions и пришлите ответ.

Проверка WB:
- /api/debug/wb больше НЕ делает live-запрос к WB по умолчанию, чтобы не тратить лимит.
- Для одного живого запроса используйте только после паузы: /api/debug/wb?live=1
- Если WB отдаёт 429, CRM сохраняет cooldown в .wb_rate_limit_until и не будет бомбить API после перезапуска.

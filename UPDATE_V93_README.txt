v93 — notifications toasts ui

Что изменено:
- Убрана выезжающая панель уведомлений, которая перекрывала интерфейс.
- Новые уведомления теперь показываются как полупрозрачные всплывающие карточки справа вверху.
- Из уведомления можно сразу открыть чат / задачу.
- Есть кнопка скрытия уведомления без перехода.
- Показываются до 4 последних непрочитанных уведомлений, остальные учитываются счётчиком.

После установки:
1. Остановить CRM.
2. Распаковать архив в C:\crm_marketplaces с заменой файлов.
3. Запустить CRM.
4. Проверить /api/debug/version — должна быть v93_notifications_toasts_ui_2026-06-18.

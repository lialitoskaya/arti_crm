v69 — кастомные статусы и воронки чатов

Добавлено:
- Новый раздел «Тех. настройки» для администратора.
- Создание/редактирование/скрытие статусов чатов.
- Создание/редактирование/удаление воронок.
- Привязка статусов к воронкам.
- Фильтр списка чатов по воронке.
- Фильтр списка чатов по кастомным статусам.
- В карточке чата статус выбирается из настроек, а не из жёсткого списка.

Системные статусы new / in_progress / waiting_customer / closed сохраняются для совместимости.
Их можно переименовать и перенести в другую воронку, но если удалить — CRM скрывает статус, а не ломает старые чаты.

Установка:
1. Остановить CRM.
2. Распаковать архив в C:\crm_marketplaces с заменой файлов.
3. Запустить CRM.
4. Проверить /api/debug/version — должна быть v69_custom_chat_statuses_funnels_2026-06-16.
5. Зайти под администратором → «Тех. настройки».
